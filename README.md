# Chat UI

A simple, static chat user interface built with HTML and Tailwind CSS.

## Description

This project provides a basic chat UI layout featuring:
- A sidebar displaying a list of chats
- A main chat window showing conversation history
- An input field for typing messages (static, no functionality)

## Features

- Responsive design using Tailwind CSS
- Sidebar with user avatars and last messages
- Chat area with message bubbles
- Input box and send button (visual only)

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)

## How to Run

1. Clone or download the project files.
2. Open `intex.html` in any modern web browser.

No server or additional setup is required as this is a static HTML page.

## Note

This is a frontend-only demo. It does not include backend functionality for sending or receiving messages. For a full chat application, integrate with a backend API.
